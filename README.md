# StudyNotion 📚

[![Live Demo](https://img.shields.io/badge/Live-Demo-blue?style=for-the-badge)](https://study-sphere-rust.vercel.app/)

StudyNotion is a modern, user-friendly web platform designed to make learning easier, more organized, and accessible.  

🚀 **Check it out here:** [StudyNotion Live](https://study-sphere-rust.vercel.app/)

---

## ✨ Features
- 📖 Clean and minimal design for better focus on learning  
- 🔍 Easy navigation to explore resources  
- 📱 Fully responsive and mobile-friendly  
- ⚡ Fast performance with modern web technologies  

---

## 🛠️ Tech Stack
- **Frontend:** React + Tailwind CSS  
- **Hosting:** Vercel  

---

## 📂 Project Setup
If you want to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/studynotion.git

# Navigate into the project folder
cd studynotion

# Install dependencies
npm install

# Run the development server
npm run dev
